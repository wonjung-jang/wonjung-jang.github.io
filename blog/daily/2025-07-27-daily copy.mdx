### Git Action을 이용한 리뷰어 자동 할당

저번 주 목요일부터였나?
회사에서 피처를 진행하다가 시간이 남아서 Git Action을 통해서 리뷰어를 자동 할당하는 기능을 만들었다.
Git Action을 테스트하기 위해서는 base 브랜치에 있는 파일을 기준으로 동작한다.
공통 기능 같은 경우에는 사내 프론트엔드 개발자 전체를 리뷰어로 지정해서 2승인 이상을 받아야 하는데, 정상 동작하지 않아 원인을 찾기 위해 로그를 추가해도 PR을 올려야 해서 부담이 됐다….
아무튼 그래도 어찌저찌 만들어서 오늘 잘 동작하는 걸 보고 슬랙 채널에 승전보를 울렸더니 뿌듯. ㅎㅎ
까먹기 전에 정리해서 블로그 글로 올려야지! ㅎㅎ
